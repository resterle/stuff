Screen commands

 * <ctrl>+a+c New window
 * <ctrl>+a+n Next window
 * <ctrl>+a+d Detach window
 * screen -r  Reattach
 * <ctrl>+a+a Switch recent two windows
 * screen -list Lists all windows

Multiscreen:

 * <ctrl>+a <shift>+s Split vertical
 * <ctrl>+a <shift>+| Split horizontal

 * <ctrl>+a <tab> Switch between sections
 * <ctrl>+a :remove Delete section
 * <ctrl>+a [  Copy mode (arrow keys to scroll)
